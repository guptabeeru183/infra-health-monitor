# Alert Runbooks

This directory contains runbooks for common alert conditions. Each file describes
steps to verify, troubleshoot, and resolve the issue, along with links to
dashboards and relevant contacts.

To add a new runbook:

1. Copy `template.md` and give it a descriptive name.
2. Fill out the sections: symptoms, impact, verification, remediation, escalation.
3. Link the file from `docs/ALERT_RUNBOOKS.md` and add annotation to relevant alert
   rule(s).
